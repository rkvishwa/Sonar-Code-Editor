# Sonar Code-Editor

## Features

- **Admin Panel**: Manage settings, view reports, and monitor user activity.
- **Collaboration Tools**: Real-time collaboration with other users, including shared editing and communication.
- **Editor Panel**: Rich code editor with syntax highlighting and file management.
- **File Tree**: Visual file explorer for easy navigation and organization.
- **Preview Panel**: Live preview of code and project output.
- **Search Panel**: Powerful search functionality across files and content.
- **Settings Modal**: Customize user and application settings.
- **Authentication**: Secure login and user management.
- **Activity Logging**: Track user actions and events for monitoring and auditing.
- **Report Generation**: Create and export reports for analysis.
- **Network Monitoring**: Monitor network status and connectivity.

## Getting Started

1. Clone the repository.
2. Install dependencies with `npm install`.
3. Start the application with `npm run dev`.

## Folder Structure

- `src/renderer/components/` - UI components
- `src/renderer/context/` - React contexts
- `src/renderer/hooks/` - Custom hooks
- `src/renderer/pages/` - Application pages
- `src/renderer/services/` - Service modules
- `src/renderer/styles/` - Global styles
- `src/renderer/types/` - Type definitions
- `src/shared/` - Shared constants and types
